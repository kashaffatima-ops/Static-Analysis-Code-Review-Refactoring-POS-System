# Static-Analysis-Code-Review-Refactoring-POS-System
A Java-based Point of Sale (POS) system reviewed and refactored as part of an SQA portfolio project. Covers comprehensive code review of 20 source files and JUnit test suites against industry-standard quality checklists. Includes applied refactoring with before-and-after comparisons, impact analysis, and documented fixes for real-world defects.
